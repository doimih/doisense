# Template: Django app

Copiază acest folder ca bază pentru un nou app Django (ex: `myapp`).
Apoi:
1. Înlocuiește `myapp` cu numele app-ului în toate fișierele.
2. Adaugă app-ul în `config.settings.INSTALLED_APPS`.
3. Adaugă URL-urile în `config.urls`.
4. Rulează `python manage.py makemigrations` și `migrate`.
